# Projet STARS

Application web pour l'association de soccer STARS

## Hébergement

L'application web a été créé avec Visual Studio Code et Laragon. On recommande leur installation pour la manipulation et le maintien des fichiers.

### Code Source

Si vous lisez ce fichier de github, vous pouvez commencer par cloner le code sur le serveur. 


Si vous utilisez Laragon, nous recommendons d'installer le fichier dans l'arboresence suivante :

    'C:\laragon\www\association_stars'

### Installer Laravel

Après que le dossier contentant les fichiers soit mis dans la bonne arborescence, ouvrez le en utilisant Visual Studio Code.
